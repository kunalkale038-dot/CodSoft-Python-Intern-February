# CODSOFT Python Internship
## Task 2 – Simple Calculator

### 📌 Project Description
The **Simple Calculator** is a Python command-line application that performs basic arithmetic operations.  
It allows users to input two numbers and select an operation to calculate the result.

This project was developed as part of the **CODSOFT Python Programming Internship**.

---

### 🎯 Features
- Perform **Addition**
- Perform **Subtraction**
- Perform **Multiplication**
- Perform **Division**
- Simple and easy-to-use **menu-driven interface**
- Handles **invalid inputs and division by zero**

---

### 🛠 Technologies Used
- **Python 3**
- Command Line Interface (CLI)
