# CODSOFT Python Internship
## Task 3 – Password Generator

### 📌 Project Description
The **Password Generator** is a Python-based command-line application that generates strong and random passwords for users.  

The user can specify the desired password length, and the program generates a secure password using a combination of **letters, numbers, and special characters**.

This project was developed as part of the **CODSOFT Python Programming Internship**.

---

### 🎯 Features
- User-defined password length
- Generates strong random passwords
- Uses letters, digits, and special characters
- Simple command-line interface
- Beginner-friendly Python implementation

---

### 🛠 Technologies Used
- **Python 3**
- Python built-in modules:
  - `random`
  - `string`
