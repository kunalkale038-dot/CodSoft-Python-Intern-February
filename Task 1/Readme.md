# CODSOFT Python Internship
## Task 1 – To-Do List Application

### 📌 Project Description
The **To-Do List Application** is a simple Python-based command-line program that helps users manage and organize their daily tasks efficiently.  

This application allows users to:
- Add tasks
- View tasks
- Update tasks
- Delete tasks

This project was developed as part of the **CODSOFT Python Programming Internship**.

---

### 🎯 Features
- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Simple command-line interface
- Beginner-friendly Python implementation

---

### 🛠 Technologies Used
- Python 3
- Command Line Interface (CLI)
