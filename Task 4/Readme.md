# CODSOFT Python Internship
## Task 4 – Rock Paper Scissors Game

### 📌 Project Description
The **Rock Paper Scissors Game** is a Python-based command-line game where a user plays against the computer.  

The user selects **rock, paper, or scissors**, and the computer randomly selects one of the three options. The program then determines the winner based on the game rules.

This project was developed as part of the **CODSOFT Python Programming Internship**.

---

### 🎯 Features
- User chooses **rock, paper, or scissors**
- Computer randomly generates its choice
- Game logic determines the winner
- Displays user choice and computer choice
- Tracks score for multiple rounds
- Option to exit the game

---

### 🛠 Technologies Used
- **Python 3**
- Python built-in module:
  - `random`
